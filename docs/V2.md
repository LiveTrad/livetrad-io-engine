# LiveTrad V2 - Expérience Vocale Améliorée 🎭

## Vue d'ensemble

La V2 se divise en deux phases majeures :
- V2.1 : Voix genrées standards
- V2.2 : Voix personnalisées

```
│   └── Voix masculine dans chaque langue

```
